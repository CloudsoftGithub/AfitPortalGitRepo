How to Obtain PIN CODES

What is PIN CODE

  1. PIN Codes are set of system generated numbers
  2. PIN codes are used for verification during registration in this portal
  3. PIN codes are only used once

How to Obtain PIN CODE

  1. To obtain PIN Codes, send a request into todunmoyebashir@gmail.com
  2. The site admin will guide you how to obtain the PIN codes
  3. The PIN codes will be sent to you after a successful verification
  4. Use the codes to signup
  5. Please, use a valid email ID during the registration process
